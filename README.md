# HighCore Agency Telegram Bot 

Welcome to the official repository for the **HighCore Agency** Telegram Bot. This bot orchestrates the HighCore Agency system natively within Telegram, effectively porting the legacy Discord infrastructure.

## Core Features
- **Client Panels**: Automated `/startup` menus to display the HighCore portfolio.
- **Ticketing System**: Allows clients to easily request services, submit inquiries, and get priority support directly via Telegram threads.
- **Automated Orders & Invoices**: Generates elegant, customized invoice and order images for transparent client transactions.
- **Feedback & Rating System**: Native 5-star rating system capturing client feedback and generating beautiful feedback images for public broadcast.
- **Dynamic Welcome Cards**: Welcomes new members with a high-quality, auto-generated golden welcome card, natively integrating their Telegram avatar and username.

## Tech Stack
- **Python 3.10+**
- **Aiogram 3.x** for asynchronous Telegram API interactions.
- **Supabase (PostgreSQL)** for reliable database management.
- **Pillow (PIL)** for rendering custom typography, shapes, and complex dynamic images (Invoices, Feedback, Welcome Cards).
